# STARBUCKS ONE PAGE

little one page to practice sass

[Live Demo](https://starbucks-onepage.vercel.app/)


Work with : 
  - React
  - Typescript
  - Sass

### Installation

#### Clone this repository
#### then
### npm
```sh
npm run install && npm run start
```
or
### yarn
```sh
yarn install && yarn start
```

For production environments...

```sh
npm run isready
```
or 
```sh
yarn isready
```
